# summer-cloud

## 功能

- 后端接口
- 后台管理系统
- 后台脚本

## 环境

- Python 3.7
- Django 2.2.25
- djangorestframework 3.12.4

## 目录

- background : 后台脚本
- base : Django 配置
- user : 用户模块
- manage.py : Django 启动

## 运行

```
python3 manage.py runserver 127.0.0.1:8000
```

## 访问

```
http://127.0.0.1:8000 首页
http://127.0.0.1:8000/admin 后台管理系统
http://127.0.0.1:8000/api 接口文档
```
